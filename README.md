# usa_printable

A printable US wall map: `usa_printable.pdf`, hand-designed, US Letter
landscape (11x8.5in). Only the finished poster is versioned here; the tooling
that produces it is not part of the public repo.

## Data credits

- **Natural Earth** 50m cultural and physical vectors
  (https://www.naturalearthdata.com/)
- **Demographia World Urban Areas** 2025
- **US Census Bureau** sub-county population estimates
- **GeoNames** cities15000
